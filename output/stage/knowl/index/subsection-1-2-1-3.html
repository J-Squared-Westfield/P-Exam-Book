Recall that a function is a set of ordered pairs \((x,y)\) where the first coordinate comes from a domain set and the second comes from a codomain set, and for each \(x\) in the domain set, there is exactly one \(y\) in the range set so that \((x,y)\) is in the function. More simply put, we can think of a function as a machine that turns inputs from a domain set into outputs in the range set. However, to be a function, each input has exactly one output. For example, if some machine \(f(x)\) had \(f(1) = 2\) and \(f(1) = -2\text{,}\) then \(f(x)\) would not be a function. As another example, if we write the equation of a circle or radius 1 centered at the origin as \(y = \pm \sqrt{1-x^2}\text{,}\) this also would not be a function (e.g.\ \(x=0\) would yield both \(y=1\) and \(y=-1\text{.}\)
