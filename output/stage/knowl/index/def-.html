Definition 1.3.7.

  • \(\lim_{x\rightarrow \infty} f(x) = L\) if for every \(\epsilon > 0\text{,}\) there is some \(x_0\) so that for any \(x\text{,}\) \(x>x_0\) implies \(L-\epsilon < f(x) < L+\epsilon.\)
