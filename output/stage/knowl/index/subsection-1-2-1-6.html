When considering inverse functions, it is helpful to recall injective or one-to-one functions. A function \(f(x)\) is one-to-one (or injective) if for any inputs \(x\) and \(y\) in the domain \(f(x) = f(y)\) implies \(x=y\text{.}\) Simply put, two different inputs must have two different outputs. For example, the function \(f(x) = 3x-1\) is one-to-one, but the function \(g(y) = x^2\) is not. (e.g. \(g(-1) = g(1) = 1\text{.}\))
