Finally, we move on to inverse functions. Let \(f\) be a function with input set \(A\) and output set \(B\text{.}\) Let \(g\) be a function with input set \(B\) and output set \(A\text{.}\) If for any \(x\in A\) and \(y\in B\text{,}\) \(y = f(x)\) precisely when \(g(y) = x\text{,}\) we call \(g\) the inverse function of f. We usually denote the inverse function of \(f\) as \(f^{-1}\text{.}\)
