Definition 1.3.2.

Let \(f(x)\) be a real-valued function and let \(a\) be a real number. We say
\begin{equation*} \lim_{x\rightarrow a} f(x) = L \end{equation*}
if for every \(\epsilon > 0\) there is some \(\delta>0\) so that for each \(x\text{,}\) \(a-\delta < x < a+\delta\) and \(x\ne a\) implies that \(L-\epsilon < f(x) < L + \epsilon\text{.}\)
