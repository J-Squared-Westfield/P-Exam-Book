Recall that \(>(x) = x^2\) is not invertible. However, the function \(>\sqrt{x}\) behaves like an inverse of it. For non-negative values of \(x\text{,}\) we have \(>\sqrt{x^2} = x\text{.}\) This is not true for negative values of \(x\text{,}\) however. But what if we only cared about the non-negative values of \(x\text{?}\) Then, in a sense, \(>x^2\) is invertible. That is, \(>x^2\) is invertible on a restricted domain.
