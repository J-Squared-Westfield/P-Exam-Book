    Section 1.2 Inverse Functions

    Do I need to review inverse functions? In a later section, we will determine the densities of functions of random variables. In doing so, we will need to work with the inverses of real- valued functions. In this section, we will briefly review inverse functions.
    🔗
    If you know how to take a non-invertible function like \(f(x) = x^2-2x-3\) and break it into invervals on which it is invertible and find inverses on each interval, you may not need to review this section.
    🔗

    Subsection 1.2.1 Fundamental Concepts

    Remember when you found out that most of the algebra you do in school was applying inverse functions to two sides of an equation and that solving \(x^2 = 4\) wasn’t as straightforward as you thought? This section serves as a refresher on that idea.
    🔗
    Recall that a function is a set of ordered pairs \((x,y)\) where the first coordinate comes from a domain set and the second comes from a codomain set, and for each \(x\) in the domain set, there is exactly one \(y\) in the range set so that \((x,y)\) is in the function. More simply put, we can think of a function as a machine that turns inputs from a domain set into outputs in the range set. However, to be a function, each input has exactly one output. For example, if some machine \(f(x)\) had \(f(1) = 2\) and \(f(1) = -2\text{,}\) then \(f(x)\) would not be a function. As another example, if we write the equation of a circle or radius 1 centered at the origin as \(y = \pm \sqrt{1-x^2}\text{,}\) this also would not be a function (e.g.\ \(x=0\) would yield both \(y=1\) and \(y=-1\text{.}\)
    🔗
    Graphically, we can tell if a curve on the Cartesian plane is a function using what is commonly called "the vertical-line test." Essentially, if it is possible to draw any vertical line that intersects your curve in more than one spot, the curve is not a function; otherwise, it is.
    🔗
    When considering inverse functions, it is helpful to recall injective or one-to-one functions. A function \(f(x)\) is one-to-one (or injective) if for any inputs \(x\) and \(y\) in the domain \(f(x) = f(y)\) implies \(x=y\text{.}\) Simply put, two different inputs must have two different outputs. For example, the function \(f(x) = 3x-1\) is one-to-one, but the function \(g(y) = x^2\) is not. (e.g. \(g(-1) = g(1) = 1\text{.}\))
    🔗
    Graphically, we can tell if a function on the Cartesian plane is one-to-one by using what is called ``the horizontal-line test.’’ Like the vertical-line test, if it is possible to draw any horizontal line that intersections the function at more than one spot, the function is not one-to-one; otherwise, it is.
    🔗

    Finally, we move on to inverse functions. Let \(f\) be a function with input set \(A\) and output set \(B\text{.}\) Let \(g\) be a function with input set \(B\) and output set \(A\text{.}\) If for any \(x\in A\) and \(y\in B\text{,}\) \(y = f(x)\) precisely when \(g(y) = x\text{,}\) we call \(g\) the inverse function of f. We usually denote the inverse function of \(f\) as \(f^{-1}\text{.}\)
    🔗
    Here is an important fact. Not all functions have inverses. If a function has an inverse, we call it invertible. Consider the function \(f(x) = x^2\text{.}\) We might think that \(g(x) = \sqrt{x}\) is its inverse. However, it is not. At least, not exactly. Recall that an inverse \(f^{-1}\) would have the requirement of being a function and \(f(x) = y\) if and only if \(f^{-1}(y) = x\text{.}\) Note that for \(f(x) = x^2\text{,}\) we have \(f(-2) = f(2) = 4\text{.}\) This would mean that an inverse of \(x^2\) would have \(f^{-1}(4) = -2\) and \(f^{-1}(4) = 2\text{.}\) This would make \(f^{-1}\) not a function. So, such an inverse does not exist. Hence, \(x^2\) is not invertible.
    🔗
    This brings up the question: when is a function invertible? It turns out, this is usually relatively easy. Recall that a one-to-one function is a function so that \(f(x_1) = f(x_2)\) implies \(x_1 = x_2\text{.}\) This would mean that for any potential inverse function \(f^{-1}\text{,}\) \(f^{-1}(y)\) would have a unique output. (Otherwise for those multiple outputs \(x_1, x_2\text{,}\) we would have \(f(x_1) = f(x_2)\text{,}\) which would make \(f\) not one-to-one.) So, \(f^{-1}\) actually is a function. From this, we make the following observation:
    🔗

    Insight 1.2.5.

    A function is invertible if and only if it is one-to-one on its domain.
    🔗
    🔗
    We proved one direction of the above observation in the preceding paragraph. The proof that invert-ability implies the function is one-to-one is similar.
    🔗
    Now onto finding inverse function. Suppose that \(f(x)\) is an invertible function. We can find the inverse function of \(f\) via the following general procedure:
    🔗
    • Begin with the equation \(y = f(x)\text{.}\)
      🔗
      🔗
    • Solve for \(x\text{.}\)
      🔗
      🔗
    • Swap the variables \(x\) and \(y\text{.}\) The resulting equation is \(y = f^{-1}(x)\text{.}\)
      🔗
      🔗
    🔗

    Example 1.2.6. Finding an Inverse.

    Find the inverse function of \(f(x) = 2x + \dfrac{1}{3}\text{.}\)
    🔗
    Answer.
    \begin{equation*} f^{-1}(x) = \frac{1}{2}x - \frac{1}{6}. \end{equation*}
    🔗
    🔗
    Solution.
    \begin{equation*} \begin{aligned} y &= 2x + \frac{1}{3}\\ y - \frac{1}{3} & = 2x\\ \frac{1}{2}y - \frac{1}{6} & = x\\ \frac{1}{2}x - \frac{1}{6} & = y \text{(swap the variables)}\\ f^{-1}(x) & = \frac{1}{2}x - \frac{1}{6} \end{aligned} \end{equation*}
    🔗
    🔗
    🔗
    Unfortunately, this process can become difficult to do by hand for more complex functions. For example, finding the inverse of \(f(x) = x^3 + x\) by hand is not particularly easy. You certainly won’t need to worry about this on the P exam.
    🔗
    There is a strong connection between a function and its inverse via function composition. We should recall composition of functions.
    🔗
    Let \(g\) be a function from input set \(A\) to output set \(B,\) and let \(f\) be a function from input set \(B\) to output set \(C\text{.}\) We call \(>f(g(x))\) or \(f\circ g\) a >composition function, which has input set \(A\) and output set \(C\) so that if \(z=f(y)\) and \(y=g(a)\text{,}\) then \(f(g(x)) = f\circ g (x) = z\text{.}\)
    🔗

    Example 1.2.7. Composition-of-Functions.

    Let \(f(x) = x^2 + 1, g(x) = \ln(5x),\) and \(h(x) = 3x-2\) on their respective domains. Determine each of the following and determine their implied domains and ranges.
    🔗
    1. \begin{equation*} f \circ g \end{equation*}
      🔗
      🔗
    2. \begin{equation*} g\circ f \end{equation*}
      🔗
      🔗
    3. \begin{equation*} h\circ f \end{equation*}
      🔗
      🔗
    4. \begin{equation*} f\circ f \end{equation*}
      🔗
      🔗
    5. \begin{equation*} f\circ g\circ h \end{equation*}
      🔗
      🔗
    🔗
    Solution.
    1. \(f\circ g = \left[\ln(5x)\right]^2 + 1.\) This has domain \((0,infty)\) and range \([1,\infty)\text{.}\)
      🔗
      🔗
    2. \(g\circ f = \ln\left[5(x^2+1)\right] = \ln(5x^2 +5)\text{.}\) This has domain \((-\infty, \infty)\) and range \([\ln(5),\infty)\text{.}\)
      🔗
      🔗
    3. \(h\circ f = 3(x^2+1) - 2 = 3x^2 + 1\text{.}\) This has domain \((-\infty,\infty)\) and range \([1,\infty)\text{.}\)
      🔗
      🔗
    4. \(f\circ f = (x^2+1)^2 + 1 = x^4 + 2x^2 + 2\text{.}\) This has domain \((-\infty,\infty)\) and range \([2,\infty)\text{.}\)
      🔗
      \(f\circ g \circ h = \left[\ln(5(3x-2))\right]^2 + 1 = \left[\ln(15x-10)\right]^2 + 1\text{.}\) This has domain \(\left(\frac{2}{3},\infty\right)\) and range \([1,\infty)\text{.}\)
      🔗
      🔗
    🔗
    🔗
    🔗
    Note that \(f^{-1}\circ f(x) = f^{-1} (y) = x\text{.}\) In a sense, the inverse function "undoes" the function. For example, consider \(f(x) = 3x-1\) and \(f^{-1}(x) = \frac{1}{3}x + \frac{1}{3}\text{.}\) These functions are inverse of each other. (We will review how to find inverse functions shortly.) Note that \(f\circ g(x) = 3\left(\frac{1}{3}x+\frac{1}{3}\right) - 1 = x\text{.}\)
    🔗
    🔗

    Subsection 1.2.2 Restricted Domains

    Recall that \(>(x) = x^2\) is not invertible. However, the function \(>\sqrt{x}\) behaves like an inverse of it. For non-negative values of \(x\text{,}\) we have \(>\sqrt{x^2} = x\text{.}\) This is not true for negative values of \(x\text{,}\) however. But what if we only cared about the non-negative values of \(x\text{?}\) Then, in a sense, \(>x^2\) is invertible. That is, \(>x^2\) is invertible on a restricted domain.
    🔗
    Consider the highly-non one-to-one function \(\sin(x)\text{.}\) Recall that it does, however, have a corresponding inverse function \(\sin^{-1}(x)\) or \(\arcsin(x)\text{.}\) How is this possible? We assume that \(\sin(x)\) has a restricted domain over which it is one-to-one. Usually, we take that restricted domain as \(\left[-\frac{\pi}{2}, \frac{\pi}{2}\right]\text{.}\)
    🔗
    Let’s return to \(>f(x) = x^2\text{.}\) Can we find an inverse for of \(f\) on the restricted domain of non-positive numbers. \(f^{-1}(x) = -\sqrt{x}\) works. (E.g.\ \(f(-3) = 9\) and \(f^{-1}(9) = -\sqrt{9} = -3\text{.}\))
    🔗

    Insight 1.2.9.

    On the P exam, when determining densities of functions of random variables, if your function is not one-to-one, you may need to break up your function onto domains in which each segment is one-to-one.

    Subsection 1.2.3 Exercises

    Checkpoint 1.2.10. Are they functions?

    Determine which of the following equations represent functions on their implied domains. Assume \(x\) is the input variable, and \(y\) is the output variable.
    🔗
    1. \(\displaystyle x=e^{y-1}-8\)
      🔗
      🔗
    2. \(x = ay-b\) for nonzero real numbers \(a,b\)
      🔗
      🔗
    3. \(\displaystyle y = \sqrt{4-x^2}\)
      🔗
      🔗
    4. \(x^2 + y^2 = r\) for \(r>0\)
      🔗
      🔗
    5. None of the above
      🔗
      🔗
    🔗
    Answer.
    \(\text{A, B, C}\)
    🔗
    🔗
    Solution.
    \(x=ay-b\) can be rearranged as \(y = \frac{1}{a}x + \frac{b}{a}\text{.}\) Since \(a\ne 0\text{,}\) this is simply a line, which is a function.
    🔗
    \(y = \sqrt{4-x^2}\) is already written in functional form. It can be seen that a single value for \(x\) with \(-2\leq x \leq 2\) will yield a single output value \(y\text{.}\) Graphically, this is the top half of a circle.
    🔗
    Solving \(x^2 + y^2 = r\) for \(y\) yields \(y = \pm \sqrt{r-x^2}\text{.}\) This is not a function. For example, \(x=0\) would give \(y = \sqrt{r}\) or \(-\sqrt{r}\text{.}\) Graphically, this is a circle with radius \(\sqrt{r}\text{.}\)
    🔗
    \(x = e^{y-1}-8\) can be rearranged as \(y = \ln(x+8) + 1\text{.}\) This is a logarithmic function.
