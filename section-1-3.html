    Section 1.3 Limits

    For the P exam, you will need to recall the basic algebraic workings of limits, particularly at infinity. Limits are also helpful in reviewing the definition of derivatives. The more advanced topics regarding limits usually seen in calculus will not be covered here such has various convergence tests as they are not needed for the P exam. We will cover a few dominance rules to provide some useful shortcuts.
    🔗

    Subsection 1.3.1 Definition of a Limit

    Let’s consider the function
    \begin{equation*} \frac{x^2+x-2}{x-1}. \end{equation*}
    Graphically, this is a line with a hole at the point \((1,3)\text{,}\) as seen below. That is, \(f(1)\) does not exist. However, the limit as \(x\) approaches \(1\) does. We see that as \(x\) values tend toward \(1\text{,}\) the function values tend toward \(3\text{.}\) In fact, the function gets as arbitrarily-close to \(3\) as we would ever like. In symbols, we would write
    \begin{equation*} \lim_{x\rightarrow 1} \frac{x^2 + x -2}{x-1} = 3. \end{equation*}
    🔗
    We give the formal definition of \(\lim_{x\rightarrow a} f(x) = L\) below.
    🔗

    Definition 1.3.2.

    Let \(f(x)\) be a real-valued function and let \(a\) be a real number. We say
    \begin{equation*} \lim_{x\rightarrow a} f(x) = L \end{equation*}
    if for every \(\epsilon > 0\) there is some \(\delta>0\) so that for each \(x\text{,}\) \(a-\delta < x < a+\delta\) and \(x\ne a\) implies that \(L-\epsilon < f(x) < L + \epsilon\text{.}\)
    🔗
    🔗
    The definition of a limit can be hard to parse. Consider the more intuitive version below.
    🔗

    Insight 1.3.3.

    \(\lim_{x\rightarrow a} f(x) = L\) means that no matter how close we want the function \(f(x)\) to be to the value \(L\) (within \(\epsilon\)), we can find a sufficiently small range of \(x\) values close to \(a\)> (within \(\delta\)) that guarantees \(f(x)\) is desirably close to \(L\text{.}\)
    🔗
    🔗
    An important note in a calculus course is that the interpretation of "\(x\) closer to \(a\) implies \(f(x)\) closer to \(L\)"" is not quite right and has some counter examples. This is why the definition of the limit is a bit more technical than many believe it should be.
    🔗
    In the following table, we provide a list of common limit rules/shortcuts. This are quite intuitive and shouldn’t be expressly memorized.
    🔗

    Common Limits.

    Suppose \(\lim_{x\to a} f(x)=L\) and \(\lim_{x\to a} g(x)=M\text{.}\) Then:
    🔗
    • For any constant \(c\text{,}\) \(\lim_{x\rightarrow a} c = c\text{.}\)
      🔗
      🔗
    • For any constant \(c\text{,}\) \(\lim_{x\rightarrow a} c\cdot f(x) = c\cdot \lim_{x\rightarrow a} f(x) = c\cdot L\text{.}\)
      🔗
      🔗
    • \(\lim_{x\rightarrow a} x = a\text{.}\)
      🔗
      🔗
    • \(\lim_{x\rightarrow a} f(x) \pm g(x) = \lim_{x\rightarrow a} f(x) \pm \lim_{x\rightarrow a} g(x) = L \pm M\text{.}\)
      🔗
      🔗
    • \(\lim_{x\rightarrow a} f(x) \cdot g(x) = \left(\lim_{x\rightarrow a} f(x)\right) \cdot \left(\lim_{x\rightarrow a} g(x)\right) = L\cdot M\text{.}\)
      🔗
      🔗
    • Suppose \(M\ne 0\text{.}\) Then \(\lim_{x\rightarrow a} \frac{f(x)}{g(x)} = \frac{\lim_{x\rightarrow a} f(x)}{\lim_{x\rightarrow a} g(x)} = \frac{L}{M}\text{.}\)
      🔗
      🔗
    • Suppose \(\lim_{x\rightarrow L} h(x) = N.\) Then \(\lim_{x\rightarrow a} h(f(x)) = \lim_{x\rightarrow L} h(x) = N\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Example 1.3.4. Limits.

    Calculate the following limits.
    🔗
    1. \begin{equation*} \lim_{x\rightarrow 3} e^{-2x+1} - 5 \end{equation*}
      🔗
      🔗
    2. \begin{equation*} \lim_{x\rightarrow \pi/2} \sin(\cos(3x)) \end{equation*}
      🔗
      🔗
    3. \begin{equation*} \lim_{x\rightarrow -3} \sqrt[4]{5x+30} \end{equation*}
      🔗
      🔗
    🔗
    Answer.
    1. \(\displaystyle e^{-5}-5\)
      🔗
      🔗
    2. \(\displaystyle 0\)
      🔗
      🔗
    3. \(\displaystyle \sqrt[4]{15}\)
      🔗
      🔗
    🔗
    🔗
    Solution.
    1. \begin{equation*} \begin{aligned} \lim_{x\rightarrow 3} e^{-2x+1} - 5 &= \lim_{x\rightarrow 3} e^{-2x+1} - \lim_{x\rightarrow 3} 5\\ &= e^{\lim_{x\rightarrow 3} -2x + 1} - 5\\ & = e^{-2 \left(\lim_{x\rightarrow 3} x\right) + \left(\lim_{x\rightarrow 3} 1\right)} - 5\\ & = e^{-2\cdot 3 + 1} - 5\\ & = e^{-5} - 5. \end{aligned} \end{equation*}
      🔗
      🔗
    2. \begin{equation*} \begin{aligned} \lim_{x\rightarrow \pi/2} \sin(\cos(3x)) &= \sin \left( \cos\left( \lim_{x\rightarrow \pi/2} 3x\right)\right)\\ &= \sin \left( \cos\left( \frac{3\pi}{2}\right)\right)\\ &=\sin(0) = 0 \end{aligned} \end{equation*}
      🔗
      🔗
    3. \begin{equation*} \begin{aligned} \lim_{x\rightarrow -3} \sqrt[4]{5x+30} &= \sqrt[4]{\lim_{x\rightarrow -3} (5x+30)}\\ &= \sqrt[4]{5 \lim_{x\rightarrow -3} x + \lim_{x\rightarrow -3} 30}\\ &= \sqrt[4]{5\cdot (-3) + 30}\\ &= \sqrt[4]{15} \end{aligned} \end{equation*}
      🔗
      🔗
    🔗
    🔗
    🔗
    Note that all of these functions are continuous around the desired \(a\) values, so we could just simply plug in the \(a\) values into the functions to get the desired limits. While there are certainly more complex and interesting limit examples, the P exam is not a calculus exam; limits are relatively straightforward.
    🔗
    Sometimes limits do not approach a finite value but instead tend to \(\infty\) or \(-\infty\text{.}\)
    🔗

    Definition 1.3.5.

    • We say \(\lim_{x\rightarrow a} f(x) = \infty\) if for every \(L\) there is some \(\delta\) so that for each \(x\) with \(0<|x-a|<\delta\) we have \(f(x) > L\text{.}\)
      🔗
      🔗
    • We say \(\lim_{x\rightarrow a} f(x) = -\infty\) if for every \(L\) there is some \(\delta\) so that for each \(x\) with \(0<|x-a|<\delta\) we have \(f(x) < L\text{.}\)
      🔗
      🔗
    🔗
    🔗
    These definitions essentially mean that for $x$ values sufficiently close to \(a\) we have the function tending toward \(\pm \infty\text{.}\)
    🔗

    Example 1.3.6.

    • \begin{equation*} \lim_{x\rightarrow 0} \frac{1}{x^2} = \infty \end{equation*}
      🔗
      🔗
    • \begin{equation*} \lim_{x\rightarrow \pi/2} -(\tan(x))^2 = \lim_{x\rightarrow \pi/2} -\left(\frac{\sin(x)}{\cos(x)}\right)^2 = -\infty \end{equation*}
      🔗
      🔗
    🔗
    🔗
    🔗

    Subsection 1.3.2 Limits at Infinity

    Most of the limits you will need to calculate on the P exam will be limits at infinity.
    🔗

    Definition 1.3.7.

    • \(\lim_{x\rightarrow \infty} f(x) = L\) if for every \(\epsilon > 0\text{,}\) there is some \(x_0\) so that for any \(x\text{,}\) \(x>x_0\) implies \(L-\epsilon < f(x) < L+\epsilon.\)
      🔗
      🔗
    • \(\lim_{x\rightarrow -\infty} f(x) = L\) if for every \(\epsilon > 0\text{,}\) there is some \(x_0\) so that for any \(x\text{,}\) \(x<x_0\) implies \(L-\epsilon < f(x) < L+\epsilon.\)
      🔗
      🔗
    🔗
    🔗
    If \(\lim_{x\rightarrow \pm \infty} f(x)\) and \(\lim_{x\rightarrow \pm \infty} g(x)\) exist and are finite, then the basic rules for limits carry over for limits at infinity. Here are some useful basic rules for limits at infinity.
    🔗

    Common Rules for Limits at Infinity.

    • For positive \(n\text{,}\) \(\lim_{x\rightarrow \infty} x^n = \infty\) and \(\lim_{x\rightarrow \infty} x^{-n} = \lim_{x\rightarrow \infty} \frac{1}{x^n} = 0\text{.}\)
      🔗
      🔗
    • For positive \(n\text{,}\) \(\lim_{x\rightarrow -\infty} x^n = -\infty\) if \(n\) is odd and \(\lim_{x\rightarrow -\infty} x^n = \infty\) if \(n\) is even.
      🔗
      🔗
    • For \(a>1\text{,}\) \(\lim_{x\rightarrow \infty} a^x = \infty\) and \(\lim_{x\rightarrow -\infty} a^x = 0.\)
      🔗
      🔗
    • For \(0<a<1\text{,}\) \(\lim_{x\rightarrow \infty} a^x = 0\) and \(\lim_{x\rightarrow -\infty} a^x = \infty.\)
      🔗
      🔗
    🔗
    🔗

    Example 1.3.8. Limits at Infinity.

    Find the following limits.
    🔗
    1. \begin{equation*} \lim_{x\rightarrow -\infty} e^{x^2 - 3x+1} \end{equation*}
      🔗
      🔗
    2. \begin{equation*} \lim_{x\rightarrow \infty} \frac{5}{7x^2} \end{equation*}
      🔗
      🔗
    3. \begin{equation*} \lim_{x\rightarrow \infty} 2^{-3x+1} \end{equation*}
      🔗
      🔗
    🔗
    Answer.
    1. \(\displaystyle \infty\)
      🔗
      🔗
    2. \(\displaystyle 0\)
      🔗
      🔗
    3. \(\displaystyle 0\)
      🔗
      🔗
    🔗
    🔗
    Solution.
    1. Since \(x^6 - 1\) tends toward infinity as \(x\) approaches \(-\infty\text{,}\) we have \(e^{x^6-1}\) tending toward \(\infty\text{.}\)
      🔗
      🔗
    2. \begin{equation*} \lim_{x\rightarrow \infty} \frac{5}{-7x^2} = -\frac{5}{7}\lim_{x\rightarrow \infty}{1}{x^2} = 0. \end{equation*}
      🔗
      🔗
    3. Since \(-7x^2\) tends toward \(-\infty\text{,}\) we have \(\lim_{x\rightarrow \infty} 5^{-7x^2} = 0.\)
      🔗
      🔗
    🔗
    🔗
    🔗
    🔗

    Subsection 1.3.3 Limits at Infinity and Dominance

    Let’s consider \(\lim_{x\rightarrow \infty} \frac{x}{e^x}\text{.}\) How might we go about finding this limit? We cannot use the basic rules given above. For these kinds of limits (often called "indeterminate"), it is handy to have some rules established in calculus. They can easily be proved using L’Hopital’s rule.
    🔗

    Dominance Rules.

    • Exponential functions "dominate" polynomial functions. This means we have the following, where \(>p(x)\) is an \(n\)-degree polynomial and \(a>\)$:
      \begin{equation*} \lim_{x\rightarrow \infty} \frac{p(x)}{a^x} = 0 \text{ and } \lim_{x\rightarrow \infty} \frac{a^x}{p(x)} = \pm \infty. \end{equation*}
      🔗
      🔗
    • Polynomial functions "dominate" logarithmic functions. This means we have the following, where \(p(x)\) is an \(n\)-degree polynomial and \(\log(x)\) is a generic logarithmic function:
      \begin{equation*} \lim_{x\rightarrow \infty} \frac{p(x)}{\log(x)} = \pm \infty \end{equation*}
      and
      \begin{equation*} \lim_{x\rightarrow \infty} \frac{\log(x)}{p(x)} = \pm \infty. \end{equation*}
      🔗
      🔗
    • Exponential functions ``dominate’’ logarithmic functions. We get similar results as above.
      🔗
      🔗
    • Higher-degree polynomials "dominate" lower-degree polynomials. Let \(p(x)\) and \(q(x)\) be (non-constant) polynomial functions with \(p(x)\) with higher degree. We have the following:
      \begin{equation*} \lim_{x\rightarrow \pm \infty} \frac{p(x)}{q(x)} = \pm \infty$ and $\displaystyle \lim_{x\rightarrow \pm \infty} \frac{q(x)}{p(x)} = 0. \end{equation*}
      🔗
      🔗
    • If \(p(x)\) and \(q(x)\) are polynomials with equal degree, with leading-term coefficients \(a\) and \(b\)>, respectively, then
      \begin{equation*} \lim_{x\rightarrow \pm \infty} \frac{p(x)}{q(x)} = \pm \frac{a}{b}. \end{equation*}
      🔗
      🔗
    🔗
    Note that in the above, the whether the limits are positive or negative depend on whether the polynomial limits tend to \(\infty\) or \(-\infty\text{.}\)
