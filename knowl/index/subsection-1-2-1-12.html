Here is an important fact. Not all functions have inverses. If a function has an inverse, we call it invertible. Consider the function \(f(x) = x^2\text{.}\) We might think that \(g(x) = \sqrt{x}\) is its inverse. However, it is not. At least, not exactly. Recall that an inverse \(f^{-1}\) would have the requirement of being a function and \(f(x) = y\) if and only if \(f^{-1}(y) = x\text{.}\) Note that for \(f(x) = x^2\text{,}\) we have \(f(-2) = f(2) = 4\text{.}\) This would mean that an inverse of \(x^2\) would have \(f^{-1}(4) = -2\) and \(f^{-1}(4) = 2\text{.}\) This would make \(f^{-1}\) not a function. So, such an inverse does not exist. Hence, \(x^2\) is not invertible.
